Feasibility Report

Features

  1. Multi-step instructions on how to use the interface
  2. Cards with prompts which can be shuffled
  3. Form submission for photo and written responses
  4. Compilation of responses into a time capsule which can be sent to a loved on
  5. Repository of responses for families

Technologies

  1. Animate on Scroll
  2. Rando.js
  3. Back4App
  4. Simple Mail Transfer Protocol aka smtp.js
  5. Back4App and JIMP

Skillset

I used the AOS, Back4App, and JIMP libraries for my Studio 3, and I feel confident using them for my capstone. I believe that I can become skilled enough to also utilized the Rando.js and smtp.js libraries as well.
